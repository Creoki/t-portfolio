Work

架空カフェサイト

Webサイト制作

プロジェクト概要

制作期間 約2週間
担当範囲 デザイン / コーディング / WordPress実装
使用技術 HTML / CSS / JavaScript / PHP / WordPress
サイトURL https://example.com

制作ポイント

この架空カフェサイトは、「親しみやすさ」と「清潔感」を意識したデザインで構築しました。 店舗情報・メニュー・アクセス情報をシンプルにまとめ、スマートフォンでも見やすいレイアウトにしています。

トップページには大きなメインビジュアルを配置し、訪問者が一目で店舗の雰囲気を感じられるように設計しました。 また、WordPressのカスタム投稿を使用して、メニューを簡単に更新できる仕組みを導入しています。
